## Functions

### Decode-Url

> Decodes a url encoded string and returns the result.

#### Syntax

```excel
Decode-Url(url)
```

Parameter | Description | Required | Type
---|---|---|--
url |The URL encoded string to decode| Yes | Text

#### Output

> Returns a decoded Text string
